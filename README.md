# slurry_viscnet
Slurry viscnet is a viscosity estimator using computer vision and deep learining methods.
This model receives video of a swirling video of a slurry, segments surface pattern frame by frame, and execute LSTM regression for slurry viscosity estimation. 
It is currently in development stage.

## Installation

```bash
$ git clone https://github.com/erstool37/slurry_viscnet
```
